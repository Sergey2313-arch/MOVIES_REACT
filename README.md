# MOVIES_REACT

Учебное React-приложение для поиска фильмов через OMDb API.

## Описание

Приложение получает данные из внешней базы фильмов OMDb API и выводит результаты поиска в виде карточек.

На карточке отображается:

- постер фильма;
- название;
- год выпуска;
- тип результата.

## Технологии

- React
- JavaScript
- CSS
- Fetch API
- OMDb API
- GitHub Pages

## Возможности

- поиск фильмов по названию;
- загрузка данных из API;
- вывод списка фильмов;
- прелоадер во время загрузки;
- обработка ошибок.

## Запуск локально

```bash
npm install
npm start
```

Приложение откроется по адресу:

```text
http://localhost:3000
```
